# Typing Speed Test Game

## Project Introduction
This project is a simple and beautiful typing speed test web game designed to help users improve their typing speed and accuracy. Users can input random sentences within a limited time, and the system will automatically calculate typing speed and accuracy.

## Main Features
- Randomly generate test sentences
- Real-time timing and progress bar display
- Calculate typing speed (WPM) and accuracy
- Error highlighting
- Result display and retest function
- Responsive design, suitable for various devices

## Usage
1. Open the `index.html` file to start the game.
2. Click the "Start Test" button and follow the prompts to enter sentences.
3. After entering, the system will automatically display your typing speed and accuracy.
4. Click "Restart" for multiple tests.

## Dependencies
- Native HTML, CSS, JavaScript, no additional dependencies required.
- Font recommendations: use local or online fonts (such as Inter Bold, Roboto Mono, Orbitron, Fira Code).

## File Structure
- `index.html`: Main page structure
- `style.css`: Page style
- `script.js`: Core logic

## How to Run
1. Download or clone this project locally.
2. Open the `index.html` file directly using a browser.

## Preview
You can deploy the project to a static server or directly open it in a local browser for experience.

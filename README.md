# 打字速度测试小游戏

## 项目简介
本项目是一个简洁美观的打字速度测试网页小游戏，旨在帮助用户提升打字速度和准确率。用户可以在限定时间内输入随机句子，系统会自动统计打字速度和正确率。

## 主要功能
- 随机生成测试句子
- 实时计时与进度条显示
- 统计打字速度（WPM）和准确率
- 错误高亮提示
- 结果展示与重测功能
- 响应式设计，适配多种设备

## 使用方法
1. 打开 `index.html` 文件即可开始游戏。
2. 点击“开始测试”按钮，按照提示输入句子。
3. 输入完成后，系统会自动显示你的打字速度和准确率。
4. 可点击“重新开始”进行多次测试。
